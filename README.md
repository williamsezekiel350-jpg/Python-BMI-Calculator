## BMI Calculator

A simple Python script that calculates Body Mass Index (BMI) and provides health category feedback based on user input.

### Features
- User-friendly input prompts for name, weight, and height
- Accurate BMI calculation using the formula: BMI = weight (kg) / height (m)²
- Health category classification:
  - Underweight (BMI < 18.5)
  - Normal Weight (18.5 ≤ BMI < 25)
  - Overweight (25 ≤ BMI < 30)
  - Obese (BMI ≥ 30)
- Personalized feedback messages based on BMI category

### Usage
Run the script and follow the prompts:
```bash
python bmi_calculator.py
